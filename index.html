Welcome to CopyAndPasted !

    This is a open source document writer hosted on GitHub. And its built entirely for the browser. 

    Try it out... [click here and start typing... or anywhere on this page for that matter].

    What you can do..

    1. Use the page editor to write structured documents with headings and different fonts
    2. Insert images into the page. 
    3. Use the browsers native console to fine tune your document. Right click + "Inspect element"
    4. Click the print button and if the browser supports it "Print as pdf"

    What you can't do as yet..

    1. Save your work for others to see (yet, although i'm working on the authentication mechanism)
    2. complain... but you can fork it and contribute back.

    What's next.

    Bug fixing, authentication and some kind of backend to make documents shareable.
